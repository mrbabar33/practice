# Dynamic To-Do App

A simple To-Do App built using HTML, CSS, and JavaScript.

## Features

- Add new tasks
- Delete tasks
- Mark tasks as completed
- Empty input validation
- Auto focus on input field

## DOM Concepts Used

- querySelector
- createElement()
- appendChild()
- addEventListener()
- classList.toggle()
- remove()

## How It Works

1. User writes a task in the input field
2. Clicks the Add button
3. JavaScript creates a new list item dynamically
4. Delete
